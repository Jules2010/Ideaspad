Quick Start
  IDEASPAD  


1.Creating new root branches

To do this move your mouse pointer over the file menu and click the left mouse button.
Then from the menu click on 'New root branch'.
You will be asked to provide a name for the branch / topic.

2. Creating new sub branch

To create a new sub branch move your mouse pointer over the branch.
Then press your right mouse button and select 'Add Sub branch'.
Then you will be asked for a name for the branch / topic.

3. Renaming a branch

To rename a branch move your mouse pointer over the branch.
Then press your right mouse button and select 'Rename branch'.

4. Deleting a branch

To delete a branch move your mouse pointer over the branch.
Then press your right mouse button and select 'Delete branch'.

5. Creating a branch link

As a means of avoiding repetition it is sometime useful to have the same topic shown under different branches.
To link to a branch drag (left click and move) your mouse pointer to where you would like the link to appear.
Then release your left mouse button.
A menu will appear, select 'Link branch here' and left click on it.

6. Changing branch icon

To change a branch icon move your mouse pointer over the branch.
Then press your right mouse button and select 'Properties', and select the 'Icons' tab.
You will now see a selection of icons, click on one and click OK.

For more information please click here.

7. Changing branch sorting

By default, branches are sorted alphabetically.  You can specify numbered ordering too.
On the menu select 'Tools' then 'Options'.  Then select the 'Sort Order' tab, and choose 'Branch sequence number'.
Then to specify sequence number for a branch, move your mouse pointer over a branch and press your right mouse button, then select 'Properties'.
Change the sequence number to a higher number to move the branch down the tree.
NB. Zero if the lowest number.

For more information please click here.

8. Printing / Saving Rich Text

You can print the current topic by clicking on the print button on the toolbar.
OR
Use the 'Selective Print' feature from the file menu.
OR
Use the 'Export' feature to save as a 'Rich Text File (RTF)'.

Here you can select certain topics and then print or export them (save them to an RTF document).

9. Using a Template

Select the 'Templates menu' then click on the 'Select Template' Option.
A 'Template Selection' screen will be shown. On the left are all the standard templates.
Select one 'Address Form' for example, now click the 'Select' button.
On the next screen select 'Create a new root topic' and click the 'OK' button.
You will be prompted to provide a name for your topic, similar to above (creating branches).

For more information please click here.
