Export Assistent
  IDEASPAD  


Der Import und Export Assistent erlaubt es Ihnen Ihre Daten in Ideaspad zwischen verschiedenen Ideaspad Datenbanken und anderen Anwendern auszutauschen.

Sie können bestimmen welche Daten exportiert oder importiert werden sollen.

   

Sie werden sich vielleicht darüber wundern, warum Sie Daten an andere Anwender weitergeben sollten. Ideaspad ist ein mächtiges Werkzeug, das Sie an Ihre Bedürfnisse anpassen können.
Vielleicht möchte ein anderer Anwender von Ideaspad genau die gleiche Konfigurationen, Vorlagen oder Notizen die auch Sie verwenden.
Darum wurde der Import und Export Assistent entwickelt.

Zum Beispiel erstellen Sie sich Ihr eigenes Buchhaltungssystem in Ideaspad.

Nachdem Sie die Daten selektiert haben, die Sie exportieren möchten, bestätigen Sie dies mit einem Klick auf die Schaltfläche "Weiter".

Danach wird Ihnen der folgende Dialog angezeigt.

   

Hier können Sie auswählen welche Knoten/Notizen exportiert werden sollen.

Klicken Sie jetzt wieder auf 'Weiter'.

Wenn Sie andere Daten (z.B. Vorlagendateien oder Farbschemata) exportieren möchten, erhalten Sie einen entsprechend angepassten Auswahldialog.

Der Assistent prüft jetzt Ihre Auswahl und informiert Sie wenn Inkonsistenzen beim Export festgestellt werden.
Dies ist zum Beispiel dann der Fall, wenn Sie Lesezeichen für einen Eintrag exportieren, der jedoch nicht im Export enthalten ist.


Aber das sind NUR Warnungen!
Stattdessen kann es auch sein, dass Sie den Eintrag bereits gelöscht haben und die Lesezeichen dann sowieso überflüssig wären.

Klicken Sie jetzt auf die Schaltfläche "Export" und geben Sie einen entsprechenden Dateinamen ein. Danach erstellt Ideaspad den entsprechenden Datenexport.
