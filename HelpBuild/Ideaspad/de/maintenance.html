Ideaspad Datenbank

Alle Informationen die Sie in Ideaspad eintragen werden in Datenbanken gespeichert..
Alle Ideaspad Datenbanken benutzen die Dateierweiterung IBF und IBW. Wenn Sie von Ihren Daten in Ideaspad eine Sicherung anlegen möchten, so müssen Sie nur diese beiden Dateien sichern.

Sie können bis zu 19 verschiedene Ideaspad Datenbanken anlegen. In der Shareware Version ist dies jedoch auf eine Ideaspad Datenbank eingeschränkt.
Diese Datenbanken können für unterschiedliche Zwecke und zur Speicherung verschiedener Datenbestände genutzt werden.

Datenpflege

Um Ideaspad Datenbanken zu erzeugen, zu löschen, umzubenennen oder zu öffnen nutzen Sie bitte den Dialog zur Datenpflege.
Sie erhalten den Dialog durch Klick auf die Schaltfläche 'Pflege' im Startdialog von Ideaspad. Bei der ersten Nutzung von Ideaspad wird dieser Dialog nicht angezeigt.

   

Die 5. Option 'Reparieren und Komprimieren' verkleinert den Speicherbedarf Ihrer Ideaspad Datenbank und führt eine Integritätsprüfung der Datenbank durch.
Ideaspad Datenbanken können mit der Zeit sehr groß werden. Aus diesem Grund empfiehlt es sich, Ihre Ideaspad Datenbanken von Zeit zu Zeit mit Hilfe dieser Funktion zu komprimieren.
