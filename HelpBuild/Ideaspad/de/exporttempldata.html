Exportiere Vorlagen
  IDEASPAD  


Der Export Assistent für Vorlagen erlaubt es Ihnen den Inhalt von Vorlagen in eine Datei zu speichern, die eventuell durch Software von Drittanbietern genutzt werden kann.

Nach dem Export können Sie solche Dateien zu Beispiel in Excel oder anderen Datenbanken verwenden.

   

Hier können Sie ganz einfach alle Daten in einer Vorlage exportieren.

Hinweis: Die Vorlagen müssen bereits in der Ideaspad Datenbank Verwendung finden, damit sie in diesem Dialog angezeigt werden.

Diese Funktion ist vor allem für professionelle Anwender von Ideaspad sinnvoll.
Sie erlaubt Ihnen den Export Ihrer Daten in ein anderes Format damit Sie auch von anderer Anwendungen bearbeitet werden können.
Eventuell haben Sie sich auch für Ideaspad entschieden, damit Sie bei der Entwicklung Ihres Informationssystems Zeit und Geld sparen.
