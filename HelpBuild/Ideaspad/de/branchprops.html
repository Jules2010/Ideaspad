Knoten Eigenschaften
  IDEASPAD  


Sie können verschiedene Eigenschaften eines Knotens ändern.
Um auf den Dialog 'Knoten Eigenschaften' zu gelangen, müssen Sie einen Knoten selektieren und danach die rechte Maustaste drücken. Wählen Sie jetzt den Menüeintrag 'Eigenschaften'.

Der nachfolgende Dialog wird angezeigt.

   

Sie können jetzt eine Sequenznummer für jeden Knoten definieren.
Dadurch können Sie die Reihenfolge ändern in welcher die Knoten im Baum angezeigt werden.
Die Sortierung nach Sequenznummern kann im Optionen Dialog im Menü Werkzeuge eingeschaltet werden.


   

Um das Symbol eines Knotens zu ändern, wählen Sie die Registerkarte 'Icons' aus. Dort wählen Sie das gewünschte Icon und bestätigen Ihre Auswahl durch Klick auf die Schaltfläche 'Übernehmen'.


In der Zukunft werden von uns voraussichtlich zusätzliche Symbole zum Verkauf angeboten. Bitte lesen Sie diesbezüglich auch unsere jeweiligen Angebote auf unserer Webseite nach.
