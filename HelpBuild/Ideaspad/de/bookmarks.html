Lesezeichen
  IDEASPAD  


Ein Lesezeichen ist eine Verknüpfung zu einer Notiz oder einem Knoten.
Wenn Sie auf ein Lesezeichen klicken wird also die entsprechende Notiz angezeigt.

Um ein Lesezeichen zu erstellen, klicken Sie entwender mit der rechten Maustaste auf einen Knoten und wählen dann 'Lesezeichen hinzufügen' oder Sie benutzen dazu den Menüpunkt 'Lesezeichen hinzufügen' aus dem Menü 'Lesezeichen'.

Nach der Auswahl wird Ihnen folgender Dialog angezeigt.

   

Geben Sie jetzt einen Namen für das Lesezeichen ein und wählen Sie ein Symbol für das Lesezeichen aus. Anschliessend klicken Sie auf OK.

Lesezeichen werden für jede Ideaspad Datenbank getrennt angelegt, aus diesem Grund besitzen Sie unterschiedliche Lesezeichen für jede Ideaspad Datenbank.

Um ein Lesezeichen zu ändern oder zu löschen, wählen Sie den Menüpunkt 'Lesezeichen verwalten' aus dem Menü 'Lesezeichen' aus.
Der nachfolgende Dialog zeigt dazu ein typisches Beispiel.

   

Ändern Sie hier entweder den Text oder das Symbol für das Lesezeichen.
