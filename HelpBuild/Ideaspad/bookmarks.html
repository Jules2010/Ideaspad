Bookmarks
  IDEASPAD  


A bookmark is a link to a topic / branch.
So once you click on the bookmark menu item, it will bring up the corresponding topic.

To create a bookmark, either, right click on the branch and select 'Add Bookmark' or click 'Add to Bookmarks' form the Bookmarks menu.

Next you will see the following screen.

   

Now type a name for your bookmark and select an Icon, then click OK.

Bookmarks are specific to each Information Base file you have, so you will have a different set of bookmarks for each.

To change or delete a bookmark, click on 'Organise Bookmarks' from the bookmarks menu.
The screen below show a typical example.

   

Alter either the text or icon, for each bookmark change!

You will see a similar maintenance screen for the template menu, so don't confuse the two!
