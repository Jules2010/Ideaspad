Template menu
  IDEASPAD  


Please see the section on templates for information about the Template Selection screen.

On the Tools menu, underneath the 'Select Template' option, you will see the 'Organise Template' menu option. Once selected the following screen will be shown.

   

You will see a similar maintenance screen for the bookmarks menu, so don't confuse the two!


In the middle of the templates menu will be some templates, providing you have added them.


At the bottom of the templates menu you will see two further features.

'Template => RTF', this allows you to change a template topic into an RTF topic, all information held in that topic will be lost, by using this feature!

'View Source', allows you to see the HTML code used in the template file.

Please see the section on creating templates for more information.
