Just Some of the Features included!

Rapid Entry New- quickly enter your information using a template
Templates New- create your own forms using HTML, some provided too.
Spell Checker - checks both rich text and templates text !!!
Rich Text Editor - a basic word processor
Tree navigation - powerful information organisation feature.

l The Power of Ideaspad
Create your own customised information collection facility or data entry system. Share your creation (theme) with others using the Export and Import features. Download templates and themes.

l What's a template?
Basically a form in which to type your information. It's made using HTML so you can write your own, with a little HTML knowledge. See Help for more information and example.

l Rich Text Editor
A basic word processor, text, colour and paragraphs. Including basic features like spell check

l Powerful information Navigation
Use the tree to organise your topics / branches into a style which suites your needs. Topics, sub topics and icons to help you identify your topic.

l Possible uses
Your own custom order entry system. A project stash. Technical information repository. Your sighting or research card file. Your own recipe catalogue. A card file for your music collection. Custom home inventory store. A birthday / Christmas card diary. Scrap book. The possibilities are endless!
